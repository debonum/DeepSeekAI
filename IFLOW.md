### 编程行为准则

#### 代码风格
```python
# ✅ 好的示例：自包含、精简、清晰
def retry_with_backoff(func, max_attempts=3, base_delay=1):
    """可直接复制使用的重试装饰器"""
    import time
    import random
    
    for attempt in range(max_attempts):
        try:
            return func()
        except Exception as e:
            if attempt == max_attempts - 1:
                raise
            delay = base_delay * (2 ** attempt) + random.uniform(0, 0.1)
            time.sleep(delay)
```

## 项目概述

DeepSeekAI 是一个基于 DeepSeek API 的非官方浏览器扩展，旨在通过智能文本分析增强网页浏览体验。用户可以通过选择网页上的文本来获取 AI 驱动的响应，使网页浏览更加高效和智能。

该项目是一个 Chrome/Firefox 浏览器扩展，支持多种 API 提供商，包括 DeepSeek 官方 API、字节跳动火山引擎、SiliconFlow 等，并提供多轮对话、Markdown 渲染、代码高亮等功能。

## 技术栈

- **前端框架**: JavaScript (ES6+)
- **构建工具**: Webpack 5
- **样式**: CSS3
- **API 集成**: OpenAI API 兼容接口
- **代码标准**: ESLint
- **包管理**: pnpm

## 常见问题

### 构建失败
- 确保已安装所有依赖：`pnpm install`
- 检查 Node.js 版本兼容性
- 清除 node_modules 重新安装

### API 连接问题
- 验证 API 密钥是否正确
- 检查网络连接
- 确认 API 提供商配置

### 扩展加载失败
- 检查 manifest.json 语法
- 确认所有必需文件都已构建
- 查看浏览器扩展管理页面的错误信息
